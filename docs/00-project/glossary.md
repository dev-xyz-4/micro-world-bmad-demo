# Glossary

## BMAD

Break, Model, Analyze, Deliver.
The structured workflow used to plan and implement changes in this project.

## EDB

Engineering Delivery Blueprint.
The delivery method and governance baseline used as the foundation for this repository.

## Micro World

A small simulation environment used here as the demo product.

## Entity

A simulated object in the micro world.
The planned core model includes position and velocity.

## Rule

A piece of logic that influences entity behavior.
Rules are planned to be explicit and composable.

## Minor Change

A local, reversible change that does not introduce new structure, new decisions, or new behavior.

## BMAD Feature

A change that introduces new structure, decisions, or behavior and therefore requires BMAD feature artifacts.
